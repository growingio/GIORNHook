# GIORNHOOK(开发中)

到工程目录下执行
`npm install --save https://github.com/growingio/GIORNHook.git`

#### Mac:
1. 打开命令行
2. 进入到node_modules/react-native-autotrack-growingio/目录下
3. 执行 `./hook.js -run`

#### Windows:
1. 在电脑上安装node
2. 打开命令行
3. 进入到node_modules/react-native-autotrack-growingio/目录下
4. 执行 `node npm -i`
5. 执行 `node ./hook.js -run`
